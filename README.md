# Academic RAG AI Agent

## Project Goals
- Build a Retrieval-Augmented Generation (RAG) agent for academic papers.
- Support PDF upload, semantic search, and natural language Q&A.

## Project Structure
- `app/` - Main Flask app
- `modelfile/` - Model configs
- `utils/` - Utility scripts
- `database/` - DB clients
- `rag/` - RAG logic
- `api/` - API endpoints
